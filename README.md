# Erasmus Mundial

A fast, mobile-first static website for **Erasmus Mundial** consisting of three static pages. Designed with an academic and global visual identity.

## 🎨 Design & Theme
- **Oxford Blue** (`#1F2A44`)
- **Parchment Cream** (`#EAE7DF`)
- **Matte Gold** (`#B08D57`)
- Clean typography and premium modern layouts.

## 📄 Pages
1. **Home (`/index.html`)**: The landing page introducing the Erasmus Mundial project.
2. **Living Languages (`/livinglanguages/index.html`)**: Information about living languages and academic programs.
3. **Evolution (`/evolution/index.html`)**: Details on the academic and project evolution.

## 🛠 Tech Stack
- **HTML5** & **Vanilla CSS**
- **Vanilla JavaScript**
- Zero external dependencies.
